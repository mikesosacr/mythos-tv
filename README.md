# MythOS TV

> Personal TV Operating System PWA

**Cinematic UI · WebOS/HarmonyOS-style · Remote Navigation · Offline-ready**

MythOS TV es una Progressive Web App (PWA) diseñada para ofrecer una experiencia de Smart TV moderna, con una interfaz cinematográfica y soporte offline.

## ✨ Características

- 🎬 Interfaz inspirada en WebOS y HarmonyOS.
- 📺 Navegación optimizada para control remoto.
- ⚡ Instalable como PWA.
- 🌐 Soporte offline mediante Service Workers.
- ❤️ Sistema de favoritos.
- 🔍 Búsqueda integrada.
- 📡 Soporte para contenido multimedia y streaming.
- 🖥️ Panel de administración integrado.
- 📱 Compatible con Smart TV, Android TV y navegadores.
- 🚀 Desarrollado en JavaScript puro (Vanilla JS).

## 🖥️ Tecnologías

- HTML5
- CSS3
- JavaScript
- Node.js
- Service Workers
- Progressive Web App (PWA)
